# 🛒 Apna Cart React App

A dynamic shopping cart application built using React.js and Bootstrap. This project demonstrates React Hooks, state management, component communication, and dynamic product handling without any backend integration.

## 🚀 Features

- Display products dynamically
- Product image support
- Increase product quantity
- Decrease product quantity
- Remove products from cart
- Real-time total amount calculation
- Responsive UI using Bootstrap
- State management with React Hooks (useState)
- Component-based architecture

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Bootstrap 5
- HTML5
- CSS3

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/onkarkarande77/Apna-Cart-ReactApp.git
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm start
```

Open:

```text
http://localhost:3000
```
